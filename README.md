- 로그인 실패 횟수(5회 이상 연속 실패)
- 동일 IP 다 계정 시도(3개 이상 계정 시도)
- 관리자 계정 실패(1회 이상 즉시 확인)

- 비 식별 IP 접속하여 정책 변경
